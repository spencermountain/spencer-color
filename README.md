# spencer-color
some colors i like to use

`npm i spencer-color`

```js
import {colors, combos} from 'spencer-color'
console.log(colors.red)
// '#cc7066'
console.log(combos.dupont)
// [ "#6accb2", "#705E5C", "#cc8a66", "#cc7066", "#7f9c6c", "#6699cc" ]
```
